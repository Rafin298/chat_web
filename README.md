## Getting Started
1. Activate the virtual environment
   
    ```
    #using git bash
    python -m venv env
    source env/Scripts/activate
    ```
2. Install the requirements
    ```
    pip install -r requirements.txt
    ```
    
3. migrate
    ```
    python manage.py makemigrations
    python manage.py migrate
    ```
